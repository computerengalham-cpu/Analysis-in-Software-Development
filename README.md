## Hospital Management System – User Stories

| ID   | User Story                                                                                       | Story Points | Priority | Acceptance Criteria                                       |
| ---- | ------------------------------------------------------------------------------------------------ | ------------ | -------- | --------------------------------------------------------- |
| H-01 | As a patient, I want to register my personal and medical details so a medical record is created. | 3            | High     | Registration saved, confirmation sent                     |
| H-02 | As a receptionist, I want to schedule appointments with doctors.                                 | 5            | High     | Appointment added, doctor/date/time confirmed             |
| H-03 | As a doctor, I want to view my scheduled patients.                                               | 3            | High     | List of today’s patients displayed                        |
| H-04 | As a patient, I want to view my medical history.                                                 | 3            | Medium   | Past visits, prescriptions displayed                      |
| H-05 | As a nurse, I want to manage hospital bed assignments.                                           | 5            | High     | Beds assigned/released, availability updated              |
| H-06 | As an accountant, I want to generate invoices for patients.                                      | 5            | High     | Invoice includes services, medications, payment processed |
| H-07 | As a hospital admin, I want to view reports on patients and revenue.                             | 5            | Medium   | Reports generated with key metrics                        |
| H-08 | As a pharmacist, I want to manage prescriptions and dispense medicine.                           | 5            | High     | Prescription processed, medicine dispensed                |
| H-09 | As a patient, I want to pay invoices online.                                                     | 3            | High     | Payment processed, receipt issued                         |
| H-10 | As a system admin, I want to create staff accounts with roles.                                   | 3            | High     | Accounts created with correct permissions                 |

## Hotel Management System – User Stories

| ID    | User Story                                                          | Story Points | Priority | Acceptance Criteria                                |
| ----- | ------------------------------------------------------------------- | ------------ | -------- | -------------------------------------------------- |
| HT-01 | As a guest, I want to book a room online.                           | 5            | High     | Booking confirmed, room assigned                   |
| HT-02 | As a receptionist, I want to check in guests.                       | 3            | High     | Booking verified, room marked occupied             |
| HT-03 | As a guest, I want to check out and pay the final bill.             | 3            | High     | Invoice generated, payment processed               |
| HT-04 | As a manager, I want to manage room types and prices.               | 3            | Medium   | Add/edit room types and prices reflected in system |
| HT-05 | As a guest, I want to pay via card or online.                       | 3            | High     | Payment processed, confirmation issued             |
| HT-06 | As housekeeping staff, I want to view rooms needing cleaning.       | 3            | Medium   | Rooms displayed, can mark as cleaned               |
| HT-07 | As a manager, I want to view occupancy and revenue reports.         | 5            | Medium   | Daily/monthly reports generated                    |
| HT-08 | As a guest, I want to book additional services (spa, room service). | 3            | Medium   | Service added to bill, confirmation sent           |
| HT-09 | As a channel manager, I want to sync availability with OTAs.        | 5            | High     | Availability updated, double bookings prevented    |
| HT-10 | As a hotel manager, I want to create user accounts with roles.      | 3            | High     | Accounts created, permissions applied              |
